# InvoiceOutPDF

LiteERP extension.

## Description

InvoiceOutPDF allows businesses to export Sales Invoices (Invoice Out) into professional PDF documents.

This extension helps users:

- Generate PDF versions of issued invoices  
- Send invoice files directly to customers  
- Maintain a consistent and clean invoice layout  
- Keep invoice documents for internal records and accounting purposes  

It improves billing professionalism and makes invoice delivery simple and efficient.

## Hooks

- Adds PDF export option to Invoice Out  
- Enables sending invoice as PDF to customers  

## Notes

This extension does not modify core modules.
