# Inventory Tracking Extension

## Overview

The **Inventory Tracking** extension allows businesses to monitor and control their inventory levels by defining a minimum stock threshold.  

When a product's available quantity reaches or falls below the configured minimum limit, the system automatically triggers a notification.

This helps prevent stockouts, reduce missed sales opportunities, and ensure timely restocking.

---

## Features

- Configure minimum inventory threshold
- Automatically receive notifications when stock reaches the defined limit
- Global minimum quantity setting
- Per-business configuration support
- Suitable for multi-business environments

---

## How It Works

1. Define the minimum inventory quantity.
2. The system continuously monitors product stock levels.
3. When stock equals or drops below the configured threshold, a notification is automatically sent.

---

## Business-Level Configuration

Each business can configure its own minimum inventory settings independently.

This allows organizations operating in a multi-business environment to apply inventory policies that match their specific operational needs.

---

## Use Cases

- Prevent stock shortages
- Improve inventory planning
- Enable proactive stock management
- Support multi-branch or multi-business operations
